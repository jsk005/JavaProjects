Java 기반 Volley Login Sample 이다.
본 코드는 Volley 라이브러리를 이용한 로그인 예제파일이다.

userID, userPW 를 단말에 저장하고, 저장된 userID, userPW로 자동로그인 시도한다.
만약 정보가 일치하지 않으면, 로그인 화면이 보인다.
단말 정보가 일치하지 않으면, 로그인 실패 팝업창이 뜬다.

서버 PHP 코드는 오픈하지 않는다.

서버 PHP 코드 기능
- 로그인 처리 : MariaDB기반 SHA256 + SALT 20자 로그인 
- 최초 로그인한 스마트폰 단말 장치번호 DB에 기록한다.
- 스마트폰 단말 장치번호가 다르면 로그인이 불가능하다.
- Android 버전 체크 기능 함수
